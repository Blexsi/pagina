Habilidades obtenidas

1Programacion de PIC

Se instruye al alumnado sobre instruciones basicas de control para un micro controlador PIC

2Implementacion de una ALU

El alumnado adquiere los conocimientos para el desarollo de una Unidad aritmetica logica usando instrucciones del data set de un PIC

3Operaciones logicas

Se adquieren las habilidades para el uso de AND,OR,XOR,NOT,etc.E implementarlos en los circuitos electronicos

4Manejo de componentes

Se obtienen las capacidades necesarias para el control de componentes electronicos como display, motores, pantallas LCD, leds , etc.
